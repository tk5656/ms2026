# ロリポップアップロード - 最初のステップ

## ステップ1: ロリポップの管理画面を開く

### 1-1. ロリポップの公式サイトにアクセス
1. ブラウザで https://lolipop.jp/ を開く
2. 右上の「ユーザー専用ページ」をクリック
3. ログイン画面で、ロリポップのユーザー名とパスワードを入力してログイン

### 1-2. サーバー管理画面を開く
1. ログイン後、契約しているサーバーを選択
2. 「サーバーの管理・設定」をクリック
3. 管理画面が開きます

**ここまでできましたか？**
- ✅ できた → ステップ2へ
- ❌ できない → ロリポップのサポートに問い合わせるか、契約情報を確認してください

---

## ステップ2: データベース情報を確認する

### 2-1. データベースメニューを開く
1. 管理画面の左メニューから「データベース」をクリック
2. データベース一覧が表示されます

### 2-2. データベースが既にあるか確認
- **既にデータベースがある場合**：
  - データベース名、ユーザー名、パスワード、ホスト名をメモしてください
  - ステップ3へ進みます

- **データベースがない場合**：
  - 「データベース作成」をクリック
  - データベース名を入力（例：`miraire`）
  - 「作成」をクリック
  - 作成されたデータベースの情報をメモしてください

### 2-3. 確認すべき情報（メモ用）
以下の情報を紙やテキストファイルにメモしてください：

```
データベース名: _______________
データベースユーザー名: _______________
データベースパスワード: _______________
データベースホスト名: _______________
（例：mysql1234.db.sakura.ne.jp）
```

**ここまでできましたか？**
- ✅ できた → ステップ3へ
- ❌ わからない → ロリポップのサポートに問い合わせてください

---

## ステップ3: 設定ファイルを編集する

### 3-1. `miraire/config.php`を開く
1. ローカルの`C:\xampp\htdocs\ms2026\miraire\config.php`を開く
2. メモ帳やエディタで開いてください

### 3-2. データベース情報を書き換える
現在の内容：
```php
const HOST = 'localhost';
const USER = 'root';
const DB = 'miraire';
const PASSWORD = '';
```

これを、ステップ2でメモした情報に書き換えます：
```php
const HOST = 'mysql1234.db.sakura.ne.jp'; // ステップ2でメモしたホスト名
const USER = 'your_username'; // ステップ2でメモしたユーザー名
const DB = 'your_database_name'; // ステップ2でメモしたデータベース名
const PASSWORD = 'your_password'; // ステップ2でメモしたパスワード
```

**例：**
```php
const HOST = 'mysql5678.db.sakura.ne.jp';
const USER = 'lolipop_user';
const DB = 'miraire_db';
const PASSWORD = 'abc123xyz';
```

### 3-3. ファイルを保存
編集したら、ファイルを保存してください。

**ここまでできましたか？**
- ✅ できた → ステップ4へ
- ❌ わからない → 具体的にどこがわからないか教えてください

---

## ステップ4: SQLファイルを実行する（データベースにテーブルを作成）

### 4-1. phpMyAdminを開く
1. ロリポップ管理画面の「データベース」メニュー
2. 「phpMyAdmin」をクリック
3. 新しいウィンドウ（タブ）でphpMyAdminが開きます

### 4-2. データベースを選択
1. 左側のデータベース一覧から、ステップ2で作成したデータベースをクリック
2. データベースが選択されます

### 4-3. SQLタブを開く
1. 画面上部の「SQL」タブをクリック
2. SQLを実行する画面が表示されます

### 4-4. SQLファイルの内容をコピー
1. ローカルの`C:\xampp\htdocs\ms2026\miraire.sql`を開く
2. ファイルの内容をすべて選択（Ctrl+A）
3. コピー（Ctrl+C）

### 4-5. SQLを実行
1. phpMyAdminのSQLタブのテキストエリアに貼り付け（Ctrl+V）
2. 画面下部の「実行」ボタンをクリック
3. 成功メッセージが表示されればOKです

**ここまでできましたか？**
- ✅ できた → ステップ5へ
- ❌ エラーが出た → エラーメッセージを確認してください

---

## ステップ5: FTP接続情報を確認する

### 5-1. FTPメニューを開く
1. ロリポップ管理画面の左メニューから「FTP」をクリック
2. FTP接続情報が表示されます

### 5-2. FTP情報をメモ
以下の情報をメモしてください：

```
FTPサーバー名: _______________
（例：ftp.lolipop.jp）

FTPユーザー名: _______________

FTPパスワード: _______________

FTPポート: _______________
（通常は21）
```

**ここまでできましたか？**
- ✅ できた → ステップ6へ
- ❌ わからない → ロリポップのサポートに問い合わせてください

---

## ステップ6: FTPソフトをインストールする

### 6-1. FileZillaをダウンロード
1. ブラウザで https://filezilla-project.org/ を開く
2. 「Download FileZilla Client」をクリック
3. Windows版をダウンロード
4. ダウンロードしたファイルを実行してインストール

### 6-2. FileZillaを起動
インストールが完了したら、FileZillaを起動してください。

**ここまでできましたか？**
- ✅ できた → ステップ7へ
- ❌ わからない → 具体的にどこで困っているか教えてください

---

## ステップ7: FTPで接続する

### 7-1. 新しい接続を作成
1. FileZillaの上部にある接続情報を入力：
   - **ホスト**: ステップ5でメモしたFTPサーバー名
   - **ユーザー名**: ステップ5でメモしたFTPユーザー名
   - **パスワード**: ステップ5でメモしたFTPパスワード
   - **ポート**: `21`（通常は21）

2. 「クイック接続」ボタンをクリック

### 7-2. 接続確認
- 接続が成功すると、右側（リモートサイト）にサーバーのファイル一覧が表示されます
- 左側（ローカルサイト）に自分のPCのファイルが表示されます

**ここまでできましたか？**
- ✅ できた → ステップ8へ
- ❌ 接続できない → FTP情報が正しいか確認してください

---

## ステップ8: ファイルをアップロードする

### 8-1. アップロードするフォルダを確認
アップロードするのは以下の2つのフォルダです：
- `miraire`フォルダ（`C:\xampp\htdocs\ms2026\miraire\`）
- `isometric`フォルダ（`C:\xampp\htdocs\ms2026\isometric\`）

### 8-2. サーバー側のディレクトリを確認
1. FileZillaの右側（リモートサイト）で、サーバーのディレクトリを確認
2. 通常は`/`または`/public_html/`が表示されます
3. このディレクトリにアップロードします

### 8-3. フォルダをアップロード
1. FileZillaの左側（ローカルサイト）で、`C:\xampp\htdocs\ms2026\`を開く
2. `miraire`フォルダを右クリック
3. 「アップロード」を選択
4. アップロードが完了するまで待ちます（時間がかかります）

5. 同じように`isometric`フォルダもアップロードします

**ここまでできましたか？**
- ✅ できた → ステップ9へ
- ❌ わからない → 具体的にどこで困っているか教えてください

---

## ステップ9: 動作確認

### 9-1. ブラウザでアクセス
1. ブラウザで以下のURLにアクセス：
   - `https://your-domain.lolipop.jp/miraire/index.php`
   - または `https://your-domain.lolipop.jp/miraire/`

2. ページが表示されれば成功です！

### 9-2. エラーが出た場合
- 「データベース接続に失敗しました」→ `config.php`の設定を確認
- ページが表示されない → ファイルが正しくアップロードされているか確認
- 画像が表示されない → `isometric`フォルダがアップロードされているか確認

---

## 困ったときは

1. **ロリポップのサポートに問い合わせる**
   - 管理画面に「サポート」や「お問い合わせ」があります

2. **エラーメッセージを確認する**
   - エラーメッセージをメモして、ロリポップのサポートに問い合わせる

3. **一つずつ確認する**
   - 上記のステップを一つずつ確認して、どこで止まっているか確認する
