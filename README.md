# ミライレ - 未来の投票所 -

## このリポジトリについて
本リポジトリは、専門学校HAL大阪にて行ったチーム制作課題として作成したWebアプリケーションです。  
オンライン投票をテーマに、「投票という行為をより身近に感じられる体験」を目的として制作しました。
